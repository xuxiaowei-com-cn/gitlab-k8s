# Nexus 简介

Nexus 是仓库管理工具，常用语搭建个人、公司私库，支持的私库类型如下：

1. hosted 代表宿主仓库，可以直接上传
2. proxy 代表代理仓库，用于搭建缓存仓库
3. group 代表分组仓库，用于合并多种仓库
